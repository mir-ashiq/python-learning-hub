# Python Learning Hub

A comprehensive, interactive Python programming learning platform built with Next.js and Tailwind CSS. Master Python from basics to advanced concepts through animated explanations, hands-on coding practice, and interactive quizzes.

## 🌟 Features

### 📚 Comprehensive Curriculum
- **Unit 1: Python Fundamentals** - Variables, data types, and basic operations
- **Unit 2: Control Structures** - Conditionals, loops, and decision making
- **Unit 3: Functions & Modules** - Code organization and reusability
- **Unit 4: Data Structures** - Lists, dictionaries, and file handling

### 🎯 Interactive Learning Components
- **Animated Concept Explanations** - Step-by-step visual learning with code examples
- **Interactive Python Practice** - Built-in code editor with instant feedback
- **Smart Quizzes** - Adaptive questions with detailed explanations
- **Data Visualization Lab** - Interactive charts and data exploration
- **Progress Tracking** - Monitor your learning journey with detailed analytics
- **User Profiles** - Personalized experience with achievements and statistics
- **Resource Hub** - Curated books, videos, and articles from Python experts

### 🚀 Technical Features
- **Next.js 14** with TypeScript for robust development
- **Tailwind CSS** for beautiful, responsive design
- **Framer Motion** for smooth animations
- **Chart.js** for data visualizations
- **Prism.js** for syntax highlighting
- **Static Export** ready for GitHub Pages deployment

## 🛠 Technology Stack

- **Frontend**: Next.js 14, TypeScript, Tailwind CSS
- **Animations**: Framer Motion
- **Charts**: Chart.js with React-Chartjs-2
- **Icons**: Lucide React
- **Code Highlighting**: Prism.js
- **Deployment**: Static export for GitHub Pages

## 🚀 Getting Started

### Prerequisites
- Node.js 18+ 
- npm or yarn package manager

### Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/mir-ashiq/python-learning-hub.git
   cd python-learning-hub
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```

3. **Run the development server**
   ```bash
   npm run dev
   ```

4. **Open your browser**
   Navigate to [http://localhost:3000](http://localhost:3000)

### Build for Production

```bash
# Build the application
npm run build

# Export static files for GitHub Pages
npm run export
```

## 🌐 Deployment

### GitHub Pages
The project is configured for static export and can be deployed to GitHub Pages:

1. Push your code to a GitHub repository
2. Enable GitHub Pages in repository settings
3. Set source to "GitHub Actions" or "Deploy from branch"
4. The site will be available at `https://yourusername.github.io/python-learning-hub`

### Other Hosting Platforms
The static export can be deployed to:
- Vercel
- Netlify
- AWS S3
- Azure Static Web Apps
- Any static hosting service
